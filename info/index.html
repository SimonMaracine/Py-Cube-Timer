Py-Cube-Timer

Py-Cube-Timer is a tool for timing your solves. If you worked with other timers before, then you'll have no problem using this.

To use this timer, you'll need to have a session created and selected. If you see nothing in the top left corner, then there is no session selected. Go ahead and create a new session, if you don't have one, by clicking File > New Session and entering a name.

To the left you have the session name, the current and best time, ao5 and ao12, the average of the entire session and a list with the solves.

To start the timer, press spacebar. If inspection is enabled, you have to press spacebar one more time. And to stop the timer, press any key.

After the timer stopped, the solve time, date and scramble are saved into the session file.

You can click on a solve (on the actual time) to see it's date and scramble and to delete it, if you want to.

You can also remove the last solve by clicking Edit > Remove Last Solve, or by pressing Alt + Z.

If you don't like the current name of your session, you can change it by clicking Edit > Rename This Session. And if you don't like how you performed at all, you can delete this session forever.

Sessions can only live in the data/sessions folder. To switch between sessions, click File > Open Session and type the name, or click Open file and select it from there.

If you want to know how you performed in your session so far, you can click File > See Statistics to see a graph of your times. It's cool!

There are some settings you can change. Click Edit > Settings. There you can change the timer and the scramble size, enable/disable inspection, change colors and enable/disable backups.

To backup your session, you have to specify a path to a writable folder you want to backup to. Every five solves, only your entire *current* session will be backed up. You can also backup manually by clicking File > Backup Session Now.

If you have a backup file of a session and you want to use it, you'll have to rename it to the session name and put it in the data/sessions folder. For example, if you have the backup file backup_2020-10_MySession, then you'll rename it to MySession.

That's all, folks.
